Q&A

Question 1: In my opinion, what are the three main differences between web1, web2, and web3?

Web1 ("readable") is more focused on the content of the website with no interaction with the users (reviews, comments and so on)
which basically means that users can't generate data on a website.

Web2 ("readable - writeable") is the next generation of websites after web1.
Unlike Web1, Web2 is designed to integrate with different users (end users, different devices, and so on).
web2 usually tries to ease the use of the website, gives the users the power to genrate data, and generally wants the Human - Computer interaction to be good.

Web3 is the future generation after web2.
web3 refers to smart use (using machine learning and data mining) of the web of each individual's data (browsing history, preferences, and so on)
in order to generate and distribute relavent useful data.

Question 2: Explain the differences by giving an example of an UseCase

Web1: The classic examples are online dictionaries which focus on the content of the website rather than the Human - Computer interaction.
Web2: Facebook is a website that represents well the Web2 concept. We can see that the website is focused on the interaction of the user with the website, and lets the user to generate data and update it (posts, photos and so on).
Web3: Tivo, a digital video recorder. Its recording program can search the web and read what it finds to you based on your preferences.
